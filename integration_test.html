DashView Light Section Integration

1. File Structure

The light section files have been added to the DashView repository:

dashview/
├── lovelace/
│   ├── light_section.yaml          # Main light section configuration
│   ├── templates/
│   │   ├── sensor_small.yaml       # Core sensor template
│   │   └── sensor_small_swipe.yaml # Swipeable sensor template
│   ├── README.md                   # Documentation
│   └── example_usage.yaml          # Usage examples
├── custom_components/dashview/     # Existing DashView integration
└── validate_yaml.py                # Validation script

2. Usage with Home Assistant

To use the light section in your Home Assistant dashboard:

Step 1: Configure Decluttering Templates

# In configuration.yaml
decluttering_templates:
  sensor_small: !include lovelace/templates/sensor_small.yaml
  sensor_small_swipe: !include lovelace/templates/sensor_small_swipe.yaml

Step 2: Include in Dashboard

# In your dashboard view
- type: vertical-stack
  cards:
    - type: custom:button-card
      name: "Living Room"
      icon: mdi:sofa
    - !include
      - lovelace/light_section.yaml
      - room: "Wohnzimmer"

3. Room Mappings

The following rooms are preconfigured with their respective light entities:

- Wohnzimmer (Living Room)    - 4 lights
- Büro (Office)              - 4 lights  
- Küche (Kitchen)            - 4 lights

- And 8 more rooms...

4. Features

  • Expandable Design: Cards can be collapsed/expanded
  • Light Counting: Shows "X von Y Lichter an" (X of Y lights on)
  • Swipe Functionality: Two-view cards showing current state and last changed
  • Multi-Device Support: Lights, switches, sensors, appliances
  • Custom Styling: CSS variables support for theming
  • Responsive Design: Works on mobile and desktop

5. Dependencies

Required Home Assistant custom components:

- custom:mod-card           # Card modification
- custom:expander-card      # Expandable sections
- custom:button-card        # Custom buttons
- custom:decluttering-card  # Template system
- custom:css-swipe-card     # Swipe functionality
Note: This light section integrates with the existing DashView system while being designed primarily for Home Assistant Lovelace dashboards. The HTML-based DashView dashboard could potentially be extended to support similar functionality using JavaScript and the Home Assistant WebSocket API.

6. Validation

Run the validation script to check YAML syntax:

python3 validate_yaml.py
